# vim
Primo modo:

Clonare il repository nella cartella home
```
$ cd ~
$ git clone https://github.com/giacfuria/vim.git
$ ln -s ~/vim/vimrc ~/.vimrc
```
Secondo modo:

Creare la cartella `vim` nella home directory e copiarci dentro il file `vimrc`
```
$ cd ~
$ mkdir vim
$ ln -s ~/vim/vimrc ~/.vimrc
```
